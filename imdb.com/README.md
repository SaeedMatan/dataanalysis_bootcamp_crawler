# IMDB SCRAPER

A web Scraper that Extracts crime genre data from IMDB

## Requirements
.Python 3.6+\
.scrapy framework\
.requests\
.pandas\
.regex

## How it works?
The scraper uses scrapy to parse the html of each page and extract the movie information. It then saves the information in a dataa.csv file.

## Output Fromat
